# 登録販売者 過去問エンジン v0.5.0

## 主な変更
- 2021〜2024年の東京都公式PDFはOCRではなくPDF内テキストを直接抽出
- 480問中478問を直接テキストで再構築
- 2022年度の問47・48のみPDF内テキスト欠落のため従来データを保持
- 問題文の不自然な文字間空白、改行、OCR文字化けを大幅削減
- 品質検査後の使用可能問題は492/600問（旧版210/600問）
- OCRは文字レイヤーのないPDFだけに使う方針へ変更

## 使い方
GitHub Pagesへ配置後、DB読込→DB検査→120問生成→統合JSON保存の順で使用します。

## 注意
2025年度は今回PDFが未提供のため従来データのままです。
